Quick Reference

Commands Every Crew Member Must Know

When the captain calls something, you respond instantly. No questions, no "wait what does that mean." Learn these before boarding.

Captain Calls
"Fenders out"Deploy now
"Lines ready"Hold, don't release
"Let go"Release that line
"Hold!"Stop everything
"All fast"Secured — done
Golden Rules
⚓ One voice runs the boat. Only the captain calls actions during maneuvers.
🚫 Never jump ashore. Step. If you can't step, boat isn't close enough.
🦺 Fenders before approach. Not during. Before.
🤫 Silence during docking. Only report hazards. Never give helm instructions.
